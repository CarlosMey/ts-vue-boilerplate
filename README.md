### Considerations
- Nodejs 16
- Vue 3
- Typescript 4.5.5

https://carlosmey.github.io/ts-vue-boilerplate/

## How to Install
```sh
npm install
npm run dev
```

### Compile and Hot-Reload for Development

```sh
npm run dev
```

### Type-Check, Compile and Minify for Production

```sh
npm run build
```

### Lint with [ESLint](https://eslint.org/)

```sh
npm run lint
```
